Package com.aerospike.client

Interface Summary
Log.Callback An object implementing this interface may be passed in to setCallback(), so the caller can channel Aerospike client logs as desired.
ScanCallback An object implementing this interface is passed in scan() calls, so the caller can be notified with scan results.

AerospikeException Aerospike exceptions that can be thrown from the client.
