Interface Summary
CitrusleafClient.LogCallback An object implementing this interface may be passed in setLogging(), so the caller can channel Citrusleaf client logs as desired.
CitrusleafClient.ScanCallback An object implementing this interface is passed in scan...() calls, so the caller can be notified with scan results.
